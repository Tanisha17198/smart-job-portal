# 💼 Smart Job Portal (AI-Based)

A full-stack web application that helps users find jobs based on their skills using an intelligent recommendation system.

---

## 🚀 Features

* 🔐 Role-based Authentication (User & Admin)
* 🤖 AI-based Job Recommendation System
* 📊 Dashboard with job match percentage
* 💼 Apply for jobs
* 🔍 Search & filter jobs
* 🛡️ Secure backend using Node.js & Express

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript

**Backend**

* Node.js
* Express.js

**Database**

* MySQL
